# go-coloring
coloring text with regexp

# USAGE

```
usage: coloring [-f file|-[rgbycpwk] regexp|-h]

        t-f file_name ... read from file instead of stdin
        t-r regexp ... to be red
        t-g regexp ... to be green
        t-b regexp ... to be blue
        t-y regexp ... to be yellow
        t-c regexp ... to be cyan
        t-p regexp ... to be purple
        t-w regexp ... to be white
        t-k regexp ... to be black
```
